# udacity-readable

#### This is a Udacity Readable Project from Udacity React Nanodegree.

To run this application, execute the following commands:

1. Run API Server:
```
$ cd api-server
$ npm install
$ node server
```
2. Run Frontend Server:
```
$ cd frontend
$ npm install (or yarn install)
$ npm start (or yarn start)
```
